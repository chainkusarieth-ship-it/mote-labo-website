# 男のモテラボ Webサイト ページ一覧（本番 v1）

制作場所：`ウェブサイト/本番/`
まだCloudflare Pagesに公開前のため、下記は**ローカルファイルパス**です。クリック（Cmd+クリック）でブラウザプレビューできます。

## ページ構成・遷移フロー

```
index.html（トップ）
  ├→ diagnosis.html（診断）→ result.html?type=xxx（結果・タイプ別）
  │                                 ├→ ［アフィリ誘導］タイプ別マッチングアプリ
  │                                 └→ line-tokuten.html（LINE登録・特典）
  │                                       └→ consultation.html（個別相談フォーム）
  └→ #video（解説動画・トップ内埋め込み）
```

## ページ一覧

| ページ | ファイル | 役割 |
|---|---|---|
| トップ（ハブ） | `index.html` | キャラ紹介・あるある共感・解説動画・SNS導線・診断への入口 |
| 診断 | `diagnosis.html` | 全8問の選択式診断（4タイプ判定） |
| 結果 | `result.html` | タイプ別結果表示、アフィリ誘導／LINE登録の2分岐（PR表記あり） |
| LINE特典 | `line-tokuten.html` | LINE公式登録の訴求・特典説明、個別相談への導線 |
| 個別相談 | `consultation.html` | 撮影プラン＋オプション選択の申込フォーム |

## 共通ファイル

| ファイル | 内容 |
|---|---|
| `style.css` | 全ページ共通スタイル（カジュアルトーン＋結果ページ以降の信頼トーン） |
| `script.js` | 診断ロジック（8問→4タイプ判定） |
| `result.js` | タイプ別結果データの描画 |
| `consultation.js` | フォームの送信処理 |

## 公開前に埋める必要がある「未確定リンク・値」一覧

サイト内に後から差し込むだけで有効化される形で仮置きしてある項目です。公開前にこれらを埋めてください。

| 項目 | 場所 | 現状 |
|---|---|---|
| YouTube解説動画のID | `index.html` の `#video` セクション | `VIDEO_ID_HERE` のプレースホルダー |
| SNSリンク（YouTube/Instagram/TikTok/X) | `index.html` の `#sns` セクション | 各`data-url`が空 |
| タイプ別マッチングアプリのアフィリリンク | `result.html` の `#appLink` | `data-url`が空（ASP登録後に発行されるリンクを設定） |
| LINE公式アカウントの友だち追加URL | `line-tokuten.html` の `#lineAddBtn` | `data-url`が空（LINE公式アカウント作成後） |
| フォーム送信先（Discord通知用） | `consultation.js` の `FORM_ENDPOINT` | 空文字（Cloudflare Worker実装後） |
